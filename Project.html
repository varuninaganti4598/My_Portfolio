PROJECT NAME:

    FARMERS CONNECT

DURATION:

    8 Weeks

DESCRIPTION:

    Farmers connect is cross platform mobile application built using react native as front end and firebase as back end. This project focuses on solving many farmer issues that they are facing in the current situations like knowing crop information, pesticides usage, fertilizers usage and etc. The team of 6 members are involved to create this app.

TECHNOLOGIES USED:

React Native - Frontend
Firebase - Backend
Expo Services
Google developer console
